# notificationBuilder
notificationBuilder[v1]
## About
Notification Builder is a Small Program Which Help You to Create Fake Notifications (For Entertainment Purposes)
## Usage
Starting Variables :
```
inf - Info Notification
wrn - Warning Notification
err - Error Notification
```
